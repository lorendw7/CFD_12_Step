# CFD 12 Step


> A Julia and Python implementation & learning notes for Computational Fluid Dynamics (CFD), adapted from the classic open-source tutorial *CFD Python: the 12 steps to Navier-Stokes equations*.

## 📖 Introduction
This repository records my personal learning process of Computational Fluid Dynamics fundamentals.

I follow the classic 12-step progressive tutorial to implement, understand and code the numerical solutions of fundamental fluid flow partial differential equations, step by step deriving from basic convection/diffusion equations to the complete Navier-Stokes equations.
